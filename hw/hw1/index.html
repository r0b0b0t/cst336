Overview

First Instance

The first instance of a computer worm is considered to be a variation of the "Creeper" program, created by Ray Tomlinson. The original iteration moved itself to different computers on the ARPANET. Ray's version, however, copied itself to different computers instead. Both iterations were not malicious in a meaningful manner; the program simply output a message.

I'm the creeper: catch me if you can!

Types

There are many types of malware, all with different ways of damaging a computer or computer network. Here is a basic overview of some types of mawlare not covered on a separate page of this website.

Virus
When executed, inserts its own code into other files, "infecting" them.
Adware
Maliciously promotes products through unwarranted pop-ups and hijacking browser searches.
Ransomware
Encrypts and holds data hostage until a payment has been made.

Security Measures

Prevention is always easier than curing. Always be wary of what is being installed; weigh the risks and benefits. Only download from reputable sources, especially if it involves software updates.
